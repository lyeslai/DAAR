Ceci sont les indications pour l'execution de notre Makefile

1/ Faire la commende "make" dans le dossier qui tme1 


2/ Pour les test on a une commande specifique pour chaque test a faire :
     
     a) pour tester l'implementation de automate sur un pattern et un texte :
        - tapez la comande "make run_automate"
        - Puis suivre les indications du programme dans la ligne de commande
    
    b) Pour tester l'implementation de KMP sur un pattern et un texte :
        -tapez la commande " make run_kmp"
        - Puis suivre les indications du programme dans la ligne de commande 

    c) Pour tester le benchmark1 qui renvoie les temps d'execution des deux mplementations sur des texte choisi aleatoirement :
        -Tapez la commande "make run_benchmark1"

    d) Pour le benchmark (2 et 3) vous faites la meme chose qu'a précédement, ie " make run_benchmark(2|3) "
